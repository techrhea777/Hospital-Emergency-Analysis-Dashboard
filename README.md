# Hospital Emergency Room Dashboard

##  Project Overview
This project presents a **Hospital Emergency Room Dashboard** built to analyze patient flow, waiting times, and overall emergency room performance.

The dashboard provides insights that help hospitals improve:
- Patient experience
- Operational efficiency
- Resource allocation

## Dashboard Features

### 🔹 Patient Analysis
- Number of patients by **age group**
- Patient distribution by **gender**
- Patient distribution by **race**

### 🔹 Operational Insights
- Admission status (Admitted vs Not Admitted)
- Patients seen within **30 minutes target**
- Department-wise referrals

### 🔹 Time-Based Analysis
- Patients by **day & hour**
- Monthly trends and performance
